# git-tools
A collection of small tools for git

## Planned features
- An easier way to install

## Installation

Add the following lines to your bashrc:
```
export GIT_TOOLS_DIR=/<path-to>/git-tools/
source $GIT_TOOLS_DIR/bashrc.sh
```
